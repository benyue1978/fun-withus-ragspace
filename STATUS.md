## 🎮 使用方法

1. **启动应用程序**
   ```bash
   source venv/bin/activate
   python app.py
   ```

2. **访问 Web 界面**
   - 打开浏览器访问 <http://localhost:8000>

3. **测试功能**
   - 上传 test_document.txt 文件
   - 在聊天界面询问关于文档的问题
   - 测试各种功能按钮
